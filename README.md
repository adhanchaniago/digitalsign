API

1. Create and Sign PDF
URL: https://pdfsign.viriyah.co.th/api/product/create
Method: POST

name = Insurance No.	
description = Description
pdf_url = Source PDF URL (http or https)
pdf_password = PDF Protect Password
category_id = Business Unit ID (1=VMI, 2=CMI, 3=NML)

Sample:

{"name":"CMI12345678","description":"Test Digital Signature API #7","pdf_url":"https://pdfsign.viriyah.co.th/test.pdf","pdf_password":"12345678","category_id":"1"}
